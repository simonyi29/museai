# MuseAI

MuseAI is a local Obsidian plugin package based on Claudian, renamed and configured as a separate plugin.

This repository contains the Obsidian-loadable packaged plugin files, not the full source project.

## Included Files

- `main.js`
- `manifest.json`
- `styles.css`

## Local Files

Local runtime settings such as `data.json`, backup files, machine-specific paths, and conversation state are intentionally ignored.

## Maintenance Note

For long-term feature work, add the original source project, package manifest, and build scripts instead of continuing to patch the bundled `main.js` directly.
